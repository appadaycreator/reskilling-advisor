# リスキリング診断【無料】あなたに向いたIT・データスキルが分かる - 技術仕様書

## 概要

**サービス名**: Reskilling Advisor
**バージョン**: 1.0.0
**更新日**: 2026-05-27
**URL**: https://appadaycreator.com/reskilling-advisor/

現職スキルと市場ニーズから最適な学び直しを診断。ITエンジニア・データサイエンス・DX人材への転換を提案。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 現在の職種・仕事内容を選択する
2. 転職・副業・昇進などのキャリア目標を選択する
3. 学習時間・PCスキル・英語力などの現状を選択する（全10問）
4. 「結果を見る」を押すとIT・語学・データ・クリエイティブ・ビジネスの5分野からスコア判定
5. 1位スキルの学習サービス・3ヶ月行動計画・2位3位候補スキルを確認する

## 診断ロジック

- 質問ごとに各スキル分野（it/language/data/creative/business）にポイントを加算
- 10問の回答スコアを合計し、最高スコアの分野を「最適スキル」として提案
- 2位・3位の候補スキルも併せて表示（複合スキル学習の参考に）

## よくある質問（FAQ）

**Q: リスキリング診断は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 社会人が今学ぶべきスキルは？**

IT・プログラミング、語学（英語）、データ分析、マーケティング、ファイナンス・会計などが需要の高いスキルです。この診断で自分に合ったスキルを見つけましょう。

**Q: リスキリングにかかる費用は？**

Udemy・YouTubeなど無料〜月1,500円で始められます。厚労省「教育訓練給付金」でスクール費用最大70%（最大56万円）が補助されます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。

**Q: 結果はどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。「今回の結果を保存」ボタンで診断日を記録し、3ヶ月後の再診断と比較できます。

**Q: 有料プランはありますか？**

いいえ、すべての機能を完全無料でご利用いただけます。
